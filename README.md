# hash-write-ai
hey, yo ! you are allowed to check your grammar mistakes and also write your content in a more human way.

---

## 🚀 Features
- 📝 Grammar correction for long text (30+ words)
- 🤖 Human-like paragraph generation
- ⚡ Fast and simple UI using Streamlit

---

## 🌐 Run the Project

### 1. Clone the Repository
git clone https://github.com/akhila-pynam/hash-write-ai.git  
cd hash-write-ai

---

### 2. Create Virtual Environment (Optional)
python -m venv .venv  
.venv\Scripts\activate

---

### 3. Install Requirements
pip install -r requirements.txt

---

### 4. Add API Key

Create a `.env` file and add:

GOOGLE_API_KEY=your_api_key_here

---

### 5. Run the Web App
streamlit run app.py

---

## 🧠 Tech Stack
- Python
- Streamlit
- Google Gemini API
- python-dotenv
